MODLINT
Report a bug

Find mod conflicts before the game does.

ModLint reads a Fabric, Forge, or NeoForge mods folder and reports what will break it: missing dependencies, version range violations, wrong-loader jars, Mixin overlaps, silent resource overrides, clashing bundled libraries, and known-bad combinations. The jars are read statically and discarded with the report.
